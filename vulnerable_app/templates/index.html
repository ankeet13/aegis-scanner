Vulnerable Test Application

WARNING: This application is INTENTIONALLY VULNERABLE.

It exists solely as a test target for the AEGIS Scanner.

DO NOT deploy this application on any public-facing server.

Pages

  • Login Page AUTH — Weak credentials, no CSRF, no rate limiting
  • Dashboard BAC — No authentication check required
  • Admin Panel BAC — No role check, shows all users
  • Product Search SQLi — SQL injection in search parameter

API Endpoints

  • GET /api/users BAC — All users, no auth required
  • GET /api/users/1 BAC — IDOR: change ID to access any user
  • GET /api/products BAC — Exposes internal notes with secrets
  • GET /api/products/3 BAC — IDOR: access non-public products
  • GET /api/orders BAC — All orders for all users
  • GET /api/search?q=widget SQLi — SQL injection via API

Exposed Files

  • /.env MISCONFIG — Database credentials, API keys
  • /.git/HEAD MISCONFIG — Git repository metadata
  • /.git/config MISCONFIG — Git remote URL
  • /config.json MISCONFIG — Application configuration with secrets
  • /robots.txt MISCONFIG — Reveals sensitive paths
  • /debug MISCONFIG — Debug info, env vars, secret key

Test Credentials

  • admin:admin — Admin role
  • test:test — User role
  • guest:guest — Guest role

How to Scan

Start the AEGIS Scanner API and React frontend, then enter http://localhost:8080 as the target URL and click Scan. The scanner will discover all endpoints listed above and flag every vulnerability.
